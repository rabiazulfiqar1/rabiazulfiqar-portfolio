
    # Rabia Zulfiqar Portfolio

    Personal portfolio website for Rabia Zulfiqar. Built with React and Vite, featuring animated sections, a custom background, and an interactive skills visualization.

    ## Highlights
    - Responsive, single-page layout with smooth section transitions
    - Interactive skills graph on desktop with a mobile-friendly fallback
    - Projects, experience timeline, and resume section with PDF viewer
    - Custom RZ favicon and social meta tags

    ## Tech Stack
    - React + TypeScript
    - Vite
    - Tailwind CSS
    - Motion (Framer Motion-compatible API)
    - Three.js

    ## Getting Started
    1. Install dependencies:
      `npm i`
    2. Start the dev server:
      `npm run dev`

    ## Build
    - Production build:
     `npm run build`

    ## Resume PDF
    Place your CV at `public/resume.pdf`. The Resume section loads and downloads this file directly.

    ## Deployment (Vercel)
    - Framework preset: Vite
    - Build command: `npm run build`
    - Output directory: `dist`
